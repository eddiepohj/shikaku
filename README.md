# Shikaku

Personal-use Shikaku puzzle PWA. Three grid sizes (7×7, 10×10, 15×15), algorithmically generated puzzles, offline-capable.

## Develop

    npm run serve   # static file server at http://localhost:8000
    npm test        # run unit tests
